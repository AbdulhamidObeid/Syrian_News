# X (Twitter) & TikTok Playbook

## 1. Algorithm & Frequency
*   **X (Twitter) Core Focus:** Breaking news, authority building, and official statements.
*   **Frequency:** 2-5 posts per day. Highly reactive to trending events.
*   **TikTok Core Focus:** Virality, hooks, and dynamic short-form video.
*   **Frequency:** 1-2 videos per day.

## 2. Tone of Voice
*   **X (Twitter):** Serious, authoritative, and extremely concise. "Thread-First Journalism" – break complex stories down into 3-4 part threads.
*   **TikTok:** Hyper-hook driven. The first 3 seconds must visually or audibly grab attention. Casual, fast-paced, and highly energetic.

## 3. Formatting
*   **X (Twitter):** Use the `portrait_1080x1350` aspect ratio for image attachments.
*   **TikTok:** Use the unified `portrait_1080x1350` aspect ratio. The Publisher agent converts this `1080x1350` image layout into a 5-second `1080x1350` MP4 video using ffmpeg before publishing it as a video post. Ensure text stays vertically centered to avoid collision with overlay UI.
