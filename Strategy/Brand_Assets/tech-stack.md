# المبادئ التوجيهية التقنية لمنصة "هاشتاق سوريا"

تحدد هذه الوثيقة القواعد التقنية لتطبيق الهوية البصرية لـ "هاشتاق سوريا" (HashSYR24) عبر كافة المنتجات الرقمية (ويب، تطبيقات هواتف، لوحات تحكم).

## قواعد الجمالية والحد الأدنى التقني (Tech Minimalist)

### 1. تطبيق الألوان (نظام الـ 9 ألوان الرسمي من Logos.ai)
- **الخلفيات والألوان المحايدة:** يجب أن تعتمد الخلفيات الأساسية على الأبيض النقي (`#FFFFFF`) والأسود المطفأ / الفحمي (`#161616`) للوضع الداكن.
- **الاستخدام الاستراتيجي للألوان البارزة:** يتم استخدام الأخضر الداكن (Deep Pine `#002D29`) والأخضر المائي (Teal `#56A596`) للعلامة التجارية الأساسية. وتستخدم درجات الأحمر (Crimson `#C93D57` و Maroon `#4A151E`) حصرياً للأخبار العاجلة والتنبيهات. 
- **اللون الذهبي:** يستخدم لون الذهبي (`#DBBE8F`) للتمييز والعناصر الفاخرة بشكل مدروس ومحدود.

### 2. قواعد الطباعة (IBM Plex Sans Arabic)
- يجب تحميل خط `IBM Plex Sans Arabic` كخط أساسي في كافة المنصات. الخطوط البديلة يجب أن تكون الخطوط العربية النظامية (`Arial`, `Tahoma`, `sans-serif`).
- **تنعيم الخطوط (Antialiasing):** يجب تفعيل تنعيم الخطوط في كافة المواقع لضمان وضوح النص:
  ```css
  body {
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
  }
  ```
- **ارتفاع السطر (Line Height):** النصوص العربية تتطلب مساحة تنفس أكبر من اللاتينية. يجب أن يكون الارتفاع الافتراضي للأسطر `1.6` أو `1.7`.

### 3. التخطيط والمساحات (Layout & Spacing)
- استخدم نظام شبكة (Grid) يعتمد على مضاعفات الرقم 8. جميع الهوامش والفراغات (Paddings & Margins) يجب أن تكون من مضاعفات 8 (مثال: 8px، 16px، 24px).
- الاستخدام السخي للمساحات السلبية (المساحات البيضاء) إلزامي لتحقيق مظهر راقي وفخم.

### 4. المكونات والتفاعلات الحركية
- **الحدود (Borders):** حدود رفيعة جداً (مثال: `1px solid #BFB4B7` للوضع المضيء أو `#3E3A3B` للوضع الداكن).
- **زوايا الحواف (Border Radius):** استخدم زوايا حادة أو دائرية بشكل خفيف جداً (`0.125rem` أو `0.25rem`). تجنب الأشكال البيضاوية المبالغ فيها.

## إعدادات Tailwind CSS

يجب على المطورين استخدام القيم المحددة في ملف `design-tokens.json` لتهيئة `tailwind.config.js`:

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        neutrals: { white: '#FFFFFF', lightGray: '#BFB4B7', darkGray: '#3E3A3B', offBlack: '#161616' },
        primary: { pine: '#002D29', teal: '#56A596' },
        accents: { gold: '#DBBE8F', crimson: '#C93D57', maroon: '#4A151E' }
      },
      fontFamily: {
        sans: ['IBM Plex Sans Arabic', 'sans-serif'],
      }
    }
  }
}
```
